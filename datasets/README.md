# Datasets

Ez a mappa tartalmazza a teszteléshez és fejlesztéshez letöltött számla-adatbázisokat.

## Tartalom

### 1. SROIE (Scanned Receipts OCR and Information Extraction)

- **Forrás:** GitHub ([https://github.com/zzzDavid/ICDAR-2019-SROIE](https://github.com/zzzDavid/ICDAR-2019-SROIE)) és Hugging Face
- **Leírás:** 1000 szkennelt nyugta annotációkkal.
- **Mappa:** `SROIE_GitHub` (GitHub verzió), `SROIE_HuggingFace` (Hugging Face verzió)

### 2. FATURA

- **Forrás:** Zenodo ([https://zenodo.org/records/8261508](https://zenodo.org/records/8261508))
- **Leírás:** 10,000 generált számla kép és JSON annotációk.
- **Mappa:** `FATURA` (kicsomagolva)

## Használat

A tesztek során ezekre a mappákra lehet hivatkozni a `datasets/` útvonalon keresztül.
