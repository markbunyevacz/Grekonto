# Számla Adatbázisok és Letöltési Linkek

## SROIE Dataset (1000 nyugta/számla)

- **Hivatalos letöltés:** [https://rrc.cvc.uab.es/](https://rrc.cvc.uab.es/) - regisztráció után elérhető a dataset
- **Kaggle:** [https://www.kaggle.com/datasets/urbikn/sroie-datasetv2](https://www.kaggle.com/datasets/urbikn/sroie-datasetv2) - 973 szkennelt nyugta annotációkkal
- **Hugging Face:** [https://huggingface.co/datasets/priyank-m/SROIE_2019_text_recognition](https://huggingface.co/datasets/priyank-m/SROIE_2019_text_recognition)
- **GitHub:** [https://github.com/zzzDavid/ICDAR-2019-SROIE](https://github.com/zzzDavid/ICDAR-2019-SROIE) - javított változat a data mappában

## FATURA Dataset (10,000 számla, 50 layout)

- **Zenodo:** [https://zenodo.org/records/8261508](https://zenodo.org/records/8261508) - 10,000 jpg kép + JSON annotációk
- **Hugging Face:** [https://huggingface.co/datasets/mathieu1256/FATURA2-invoices](https://huggingface.co/datasets/mathieu1256/FATURA2-invoices) - FATURA2 változat

## MIDD Dataset (630 PDF számla)

- **Zenodo:** [https://zenodo.org/records/5113009](https://zenodo.org/records/5113009) - Multi-Layout Invoice Document Dataset, IOB formátumú annotációkkal

## Kaggle Gyűjtemények

- **Invoices Dataset:** [https://www.kaggle.com/datasets/cankatsrc/invoices](https://www.kaggle.com/datasets/cankatsrc/invoices) - Python Faker-rel generált adatok
- **Invoices for Document AI:** [https://www.kaggle.com/datasets/holtskinner/invoices-document-ai](https://www.kaggle.com/datasets/holtskinner/invoices-document-ai)

## Roboflow Universe

- **Browse Datasets:** [https://universe.roboflow.com/search?q=class%3Ainvoice](https://universe.roboflow.com/search?q=class%3Ainvoice) - több annotált számla-dataset böngészőben
- **US Invoice Dataset:** [https://universe.roboflow.com/national-yang-ming-chiao-tung-university/us-invoice](https://universe.roboflow.com/national-yang-ming-chiao-tung-university/us-invoice) - objektumdetektálásra annotálva

## BuDDIE Dataset

- **ArXiv:** [https://arxiv.org/abs/2404.04003](https://arxiv.org/abs/2404.04003) - 1,665 valódi üzleti dokumentum multi-task információkinyeréshez

*Megjegyzés: Mind a SROIE, FATURA és MIDD nyílt hozzáférésű és ingyenesen letölthető kutatási célokra. A Kaggle és Hugging Face platformok közvetlenül letöltést vagy API-n keresztül történő hozzáférést biztosítanak.*
